# 📖 [Inventory-management] <a name="about-project"></a>

**[Inventory_management]** is a basic CRUD application that allows users to manage their inventory.

<!-- Features -->

### Key Features <a name="key-features"></a>

- you can add a new item to the inventory
- you can edit an item in the inventory
- you can delete an item from the inventory
- you can view all items in the inventory
- you can view a single item in the inventory
- you can see out of stock items in the inventory

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [Node.js](https://nodejs.org/en/download/)
- [npm](https://www.npmjs.com/get-npm)
- [MongoDB](https://docs.mongodb.com/manual/installation/)
- [Git](https://git-scm.com/downloads)

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com:Haadiiii/inventory-management.git
```

### Install

Install this project with:

```sh
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm run dev || npm start
```
